# [IT Cert Review] รีวิวเดือด! สอบ IT Cert 4 ใบใน 2 เดือน (เฉลี่ย 2 อาทิตย์ 1 ใบ) ทำได้ไงมาดูกัน

สวัสดีครับทุกท่าน ผมชื่อจิรายุ เรียกว่า **“แจ๊ค”** ก็ได้ ปัจจุบันทำงานอยู่ที่ธนาคารอาคารสงเคราะห์ (สำนักงานใหญ่) ตำแหน่ง **System Engineer**

วันนี้อยากมาแชร์ประสบการณ์สอบเซอร์ ในช่วง 2 เดือนที่ผ่านมา กับภารกิจปั้นโปรไฟล์สาย Digital Infrastructure & Cybersecurity แบบก้าวกระโดด

หลายคนถามว่า *"สอบไปทำไมเยอะแยะในเวลาแค่นี้?"* คำตอบคือการสร้าง **Foundational Knowledge** ที่แข็งแกร่งครับ เพราะใบเซอร์เหล่านี้คือ **Gold Standard** ที่การันตีว่าเราคุยกับคนในวงการรู้เรื่องและทำงานได้จริง อีกอย่างองค์กรผมสนับสนุนเรื่องการ Training เต็มที่ อะไรที่คิดว่ามีประโยชน์ต่อนาน สามารถวางแผนของบเพื่อไปเรียน+สอบได้เลยแบบไม่มีกั๊ก ทั้งนี้ต้องขอบคุณธนาคารอาคารสงเคราะห์มากๆ ครับที่สนับสนุน (เอิ่ม…คือจริงๆ ผมไป Training มาตั้งแต่ปีที่แล้ว เรียนจนคืนอาจารย์ไปเกือบหมดแล้ว 555 แต่เพิ่งจัดสรรเวลาได้มาสอบช่วงนี้)

## ⚔️ ทำความรู้จัก "ค่าย" ยักษ์ใหญ่ที่ผมไปรบมา
ทำไมถึงเรียกว่ารบ? ผมจำมาจากพ่อผม พ่อผมบอกว่า *“ไปสอบก็เหมือนไปรบนั่นแหละ”* หากเชื่อเรื่องฤกษ์ยาม ก็เลือกวันสอบที่เป็นเพชรฆาตฤกษ์ไปเลย 555 เอาล่ะมาดูกัน

* **CompTIA:** เจ้าพ่อวงการ IT Cert เน้นความรู้แบบ **Vendor-Neutral** (ไม่ยึดติดยี่ห้อ) แปลว่าความรู้นี้เอาไปใช้กับ Cisco, Microsoft, HP หรือ Cloud เจ้าไหนก็ได้หมด เป็นเซอร์ที่เป็นมาตรฐานเลย คนเล่นเขารู้กัน
* **ISC2:** สถาบันระดับโลกที่เน้นเรื่องความมั่นคงปลอดภัยทาง Cyber โดยเฉพาะ ใครอยากไปสาย Cybersecurity แบบจริงจัง ค่ายนี้คือจุดเริ่มต้นที่เป็นมาตรฐานและน่าเชื่อถือสุดๆ เซอร์ตัวชูโรงของค่ายนี้คือ **CISSP** เรียกได้ว่าเป็นเซอร์ที่ทุกคนอยากครอบครอง

## 🔍 เจาะลึก 4 ใบที่สอบผ่าน

### 1. ISC2 CC (Certified in Cybersecurity)
* **Level:** Entry-level ที่มาแรงที่สุด
* **Focus:** เน้น "วิธีคิด" แบบคนทำงานสาย Security จริงๆ เช่น Risk Management, Incident Response และ Network Security
* **Insight:** เหมาะมากสำหรับคนอยากปูพื้นฐานจรรยาบรรณและหลักการความปลอดภัยเบื้องต้น
* *หมายเหตุ: ใบนี้ใครเก่งภาษาอังกฤษจะได้เปรียบมาก เพราะเขาชอบใช้คำยากๆ ที่ไม่ได้แปลได้ตรงๆ ชนิดที่ว่าต้องแปล Eng-to-Eng อีกที (ใบนี้ผมสอบเอง ไม่มีสปอนเซอร์)*

### 2. CompTIA Security+ (SY0-701)
* **Level:** Intermediate (มาตรฐานที่ HR ทั่วโลกมองหา)
* **Focus:** เข้มข้นขึ้นมาอีกขั้น เน้นการรับมือกับ Attacks & Vulnerabilities และการวางระบบ Architecture ให้ปลอดภัยแบบ End-to-End
* **Insight:** เป็นใบที่ช่วยอัปค่าตัวได้จริง เพราะครอบคลุมเนื้อหาที่ใช้ทำงานในหน้างาน Security ปัจจุบัน

### 3. CompTIA Cloud+ (CV0-003)
* **Level:** Intermediate
* **Focus:** เจาะลึกการบริหารจัดการทรัพยากรบน Cloud, การเลือก Model ที่เหมาะสม, การทำ Migration (ย้ายระบบ) และการ Troubleshooting บน Infrastructure ยุคใหม่
* **Insight:** ใบนี้จะเน้นเคสสมมติเยอะ โจทย์จะแนววิเคราะห์ยาวๆ ต้องเข้าใจ Logic ของ Cloud จริงๆ ถึงจะผ่านครับ

### 4. CompTIA Server+ (SK0-005)
* **Level:** Intermediate
* **Focus:** บริหารจัดการ Enterprise Infrastructure ทั้งระบบ ตั้งแต่การเลือก Hardware, Software, การทำ Storage (RAID), ไปจนถึงการทำ Virtualization และการกู้คืนระบบเมื่อเกิดวิบัติภัย (Disaster Recovery)
* **Insight:** ใบนี้เน้นเรื่องการ Troubleshooting มากขึ้น รวมถึงมีเรื่อง Cloud และ Security มากขึ้นด้วย ตั้งแต่ Physical ไปถึง OS เลย

## 📖 เกร็ดความรู้เพิ่มเติม

* **ISC2 CC:** เวลาสอบ 120 นาที ประมาณ 100 ข้อ คะแนนผ่าน 700/1000 มีทั้งแบบ Multiple Choice และ Choose Two/Three สอบเสร็จบอกแค่ผ่านหรือไม่ผ่าน ไม่มีบอกคะแนน
* **CompTIA:** เวลาสอบ 90 นาที ประมาณ 75–90 ข้อแล้วแต่วิชา คะแนนผ่าน 750/1000 ข้อสอบ 1–3 ข้อแรกเป็นแบบ **PBQs (Performance-Based Questions)** เน้นวิเคราะห์และแก้ปัญหาจากสถานการณ์สมมติ และมีทั้งแบบ Multiple Choice และ Choose Two/Three สอบเสร็จบอกคะแนนด้วย (อันนี้ดี)

> 💡 **Extra Bonus:** พอสอบผ่านคอมโบนี้ ผมได้รับ **Stackable Certification** อย่าง **CSCP (CompTIA Secure Cloud Professional)** มาประดับโปรไฟล์เพิ่มด้วย! เพราะเรามีทักษะทั้ง Cloud และ Security ควบคู่กัน
>
> และผมเคยสอบผ่าน **CompTIA Linux+** มาก่อนแล้ว จริงๆ ต้องหมดอายุสิ้นปีนี้ แต่เมื่อผมสอบ **CompTIA Cloud+** ผ่าน ก็เลยได้ **Renew Linux+** โดยอัตโนมัติ เนื่องจาก Cloud+ เป็นเซอร์ที่ระดับสูงกว่า

## ⭐️ เทคนิคการสอบ: สอบผ่านใน 2 อาทิตย์ (AI-Assisted Learning)

การจะยัดเนื้อหาเล่มหนาๆ ระดับ 400–600 หน้าต่อวิชาเข้าหัวใน 14 วัน เป็นอะไรที่ยากมากสำหรับผม ผมเลยใช้วิธีนี้:

1.  **7 วันแรก (สร้างพื้นฐาน):** ผมใช้ AI เป็น "ติวเตอร์ส่วนตัว" โดยใช้ **NotebookLM + Gemini / ChatGPT** ผมโยน Student Guide เข้าไปใน NotebookLM เพื่อทำสรุปประเด็นสำคัญ เน้นสัดส่วนที่ออกสอบ ผมเชื่อว่าจะสอบผ่านนั้นพื้นฐานต้องแน่นก่อน
2.  **ติวเข้มด้วย Prompt:** ใช้ Gemini/ChatGPT ช่วยติวโดยใช้ Prompt ว่า: *"จงสวมบทบาทเป็น Exam Expert ช่วยวิเคราะห์ข้อสอบข้อนี้ และอธิบายแนวคิดแบบตัดช้อยส์ให้หน่อยว่าทำไมข้อ A ถึงถูก และ Logic ของข้ออื่นๆ ที่ผิดคืออะไร"* วิธีนี้ช่วยให้เราไม่แค่ "จำ" แต่ "เข้าใจ" ครับ
3.  **ตะลุย Exam Dumps อย่างมีชั้นเชิง:** ใช้แหล่งข้อมูลอย่าง **ExamTopics, ExamPrepper และ Braindumps** เพื่อคุ้นเคยกับทรงข้อสอบ (บางข้อมีในข้อสอบจริงด้วย)
    * **ทริคสำคัญ:** อย่าแค่จำคำตอบ! ให้เข้าไปอ่านในส่วน **Discussion** เพราะจะมีผู้เชี่ยวชาญมาถกกัน จุดนี้แหละคือคลังความรู้ที่ทำให้เราสอบผ่านได้จริง
    * **Cross-check:** ผมเจอเฉลยมั่วๆ หลายข้อเลย ผมจึงใช้ Gemini/ChatGPT ช่วยตรวจสอบคำตอบไปอีกชั้นนึงครับ

**ทริคส่วนตัว:**
* จัดสรรเวลาสอบให้ดี ข้อไหนดูแล้วยังไม่มั่นใจ ให้ **Flag** ไปทำข้ออื่นก่อน
* ความคิดแว่บแรกมักจะถูกเสมอ แต่ถ้ายังไม่มั่นใจให้กดเลือกตอบไปก่อน แล้วมา Review Flag อีกทีได้
